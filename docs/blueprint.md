# **App Name**: HiveMind

## Core Features:

- User Authentication: Secure login and registration flows for managing user accounts.
- Hive Dashboard Overview: A centralized dashboard displaying all registered beehives and their current operational status (online/offline).
- Real-time Sensor Data Display: View live data for temperature, humidity, hive weight, and bee activity for individual beehives.
- Historical Data Charts: Interactive charts to visualize historical sensor data over customizable periods (daily, weekly, monthly).
- New Device Registration: Allow users to easily register new IoT beehive monitoring devices and assign them to their accounts.
- Notification System for Alerts: Backend system to monitor sensor data against predefined safe ranges and trigger alerts for out-of-range readings (e.g., temperature, humidity).
- MQTT Data Ingestion & Storage: A robust backend system to receive and process real-time JSON sensor data from ESP32 devices via MQTT, storing it in the database for display and analysis.

## Style Guidelines:

- Primary color: A rich, earthy gold (#B87E2E) to evoke the warmth of honey and the natural environment of bees, providing a solid grounding for data.
- Background color: A serene, lightly warmed off-white (#F5F3F0) to maintain a clean, calm, and minimalist aesthetic, allowing data to stand out.
- Accent color: A warm, soft coral (#E05E52) chosen to create visual contrast for call-to-action buttons, alerts, and highlight important information without being overly bright, drawing attention analogously to the primary hue.
- Headline and body font: 'Inter', a neutral and highly readable sans-serif, ensuring a modern, objective, and minimalist feel across all text.
- Clean, minimalist line-art icons that visually represent beehives, data points, alerts, and IoT devices, aligning with the calm and modern design ethos.
- A clean, center-aligned layout on larger screens that emphasizes readability and reduces visual clutter. The UI will be fully responsive and optimized for a seamless experience on mobile devices.
- Subtle and smooth transitions for data updates and navigation, ensuring a non-disruptive user experience that complements the minimalist and calm visual design.
